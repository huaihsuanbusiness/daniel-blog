# Daniel Blog

A bilingual personal publishing site built with Astro and TypeScript.

Calm, fast, and intentionally minimal — designed for writing, ideas, and a refined multilingual reading experience.

**Live**  
https://danielcanfly.com/

**WWW**  
https://www.danielcanfly.com/

**Repository**  
https://github.com/huaihsuanbusiness/daniel-blog

---

## Stack

- Astro
- TypeScript
- Cloudflare
- GitHub

---

## Run Locally

```bash
npm install
npm run dev
```

---

## Build

```bash
npm run build
npm run preview
```

---

## Deploy

Production deploys automatically from the `main` branch through Cloudflare.
